# 🌌 Cosmos Space Dashboard

An interactive, responsive, and pixel-perfect front-end dashboard designed to explore real-time cosmic data. This project seamlessly integrates multiple open-source REST APIs to deliver continuous astronomical insights, dynamic satellite launch tracking, and in-depth planetary data analytics within a modern, user-centric interface.

Built as part of an advanced front-end assignment to master **Asynchronous JavaScript**, **API integration**, and defensive programming techniques.

---

## 🚀 Key Features

* **Today in Space (NASA APOD):** Fetches NASA's Astronomy Picture of the Day dynamically using an API key, parsing rich descriptions and media formats.
* **SpaceDevs Launches Tracker:** Tracks the top 10 upcoming global rocket launches, displaying a highlighted 'Featured Launch' along with a responsive grid layout for remaining schedules.
* **Solar System Encyclopedia:** A robust data-driven section allowing users to interactively view deep scientific metrics (gravity, mass, density, orbital speeds) for planets upon selection.
* **Robust Error Handling:** Implements defensive programming principles (`if` conditional checking, optional chaining, and status validation) to prevent UI crashes during network downtime.

---

## 🛠️ Tech Stack & Frameworks

* **Markup & Core:** HTML5, Modern Semantic Architecture
* **Styles & Layout:** CSS3, Tailwind CSS (Utility-first styling for fluid responsiveness)
* **Logic & Engine:** Vanilla JavaScript (ES6+), Fetch API, Async/Await asynchronous handling, and Dynamic DOM Manipulation
